# Multi-Modal Equity Intelligence Terminal & Predictive Risk Engine 📊🔮

An institutional-grade, data-driven financial analytics platform designed to synthesize real-time market data, structural accounting forensics, statistical risk horizons, and contextual AI intelligence into a singular decision-making dashboard.

Developed as an engineering-focused solution to decouple retail market noise from actual financial health indicators.

## 💡 Core Engineering Pillars & Features

### 1. Advanced Structural Accounting Forensics
* **Piotroski F-Score Audit:** A dynamic 9-point financial fitness execution check inspecting balance sheet leverage deceleration, liquidity growth, and margin expansion.
* **Beneish M-Score Engine:** Full mathematical implementation of the 8-variable model to flag statistical probabilities of corporate earnings manipulation (Threshold ceiling locked at `-1.78`).
* **Insolvency Tracking:** Embedded Altman Z-Score modeling to evaluate macro bankruptcy risks.

### 2. Market Expectations Investing Framework
* **Reverse DCF Loop Optimization:** Back-calculates the exact 10-year Free Cash Flow CAGR currently priced into an equity asset, giving investors an objective look at market hype versus realistic growth.
* **Historical Multiple Reversion:** Compares trailing price-to-earnings ratios against 3-5 year moving medians to isolate systemic overvaluation anomalies.

### 3. Machine Learning Ensemble Forecaster & Risk Horizon
* **Multi-Modal Predictive Signal:** Synthesizes technical momentum indicators with fundamental financial structures to train an ensemble pipeline model with localized feature-importance tracking.
* **Monte Carlo Risk Projection Cones:** Runs 1-year future asset price path projections using Geometric Brownian Motion parameters, condensed into clean Maximum, Minimum, and Expected Central Trajectory boundary limits with zero legend noise.

### 4. Context-Aware AI Research Assistant
* **State-Aware Financial Chat:** A multi-turn conversation layer reading active session corporate parameters to answer complex questions regarding capital allocation profiles, financial moats, and margin safety metrics.

## ⚙️ Local Infrastructure Setup

To spin up a clone of this terminal on your local environment:

1. Clone this repository to your machine:
   ```bash
   git clone [https://github.com/YOUR_GITHUB_USERNAME/DS_Portfolio_Project.git](https://github.com/YOUR_GITHUB_USERNAME/DS_Portfolio_Project.git)
   cd DS_Portfolio_Project

2. Install the production-grade dependencies:
    pip install -r requirements.txt

3. Configure your local environment variables. Create a .streamlit/secrets.toml file to manage your API securely:
    GEMINI_API_KEY = "YOUR_REAL_API_KEY_HERE"

4. Launch the application server:
    streamlit run app.py
